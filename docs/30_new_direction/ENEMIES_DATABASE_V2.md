# 1. Enemy Philosophy

Enemies are not only combat encounters.

They are progression nodes.

Each enemy family must provide:

```txt
combat challenge
resource generation
crafting support
lore representation
```

Every enemy belongs to:

```txt
Family
→ Variant
→ Elite Variant
→ Boss Ecosystem
```

---

# 2. Enemy Categories


Purpose:

```txt

```

Story-critical encounter.

```txt
chapter progression
```
