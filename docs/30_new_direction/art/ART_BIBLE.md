# 1. Vision Artistique

## Elevator Pitch

IdleKing est un Action RPG Dark Fantasy racontant l'histoire d'un roi oublié qui erre dans un rêve mourant afin de réparer le Fleuve de Vie.

Le jeu parle :

* du deuil
* de la mémoire
* de l'amour
* de la reconstruction
* du temps
* du sacrifice

Le ton général n'est pas héroïque.

Le joueur n'est pas un élu.

Il est un homme perdu cherchant une vérité qu'il a oubliée.

---

# 2. Piliers Émotionnels

Les émotions dominantes du projet sont :

## Solitude

Le monde semble abandonné.

Les villages sont vides.

Les routes sont mortes.

Le vent parle plus souvent que les habitants.

## Mystère

Le joueur comprend progressivement :

* qui il est
* ce qu'est le Monde Rêvé
* qui est Tobo
* qui est Waqt
* pourquoi les Ères existent

## Épopée

Malgré sa solitude, le voyage du Roi influence le destin du monde entier.

# 4. Style Graphique

## Caméra

Vue isométrique 3/4.

Distance moyenne.

Référence :

Alabaster Dawn.

Le joueur reste lisible.

Les boss peuvent occuper une large portion de l'écran.

## Niveau de réalisme

Référence :

Children of Morta.

Personnages :

* proportions réalistes
* têtes proportionnées
* silhouettes lisibles

Pas de chibi.

Pas de caricature.

## Animation

Priorité :

Fluidité.

Peu de détails inutiles.

Animation riche.

Lecture immédiate.

# 5. Le Vieux Roi

## Concept

Le Vieux Roi est un souverain déchu.

Pas par faiblesse.

Par oubli.

Il est puissant.

Mais il ignore encore pourquoi.

## Apparence

Âge apparent :

30 ans

Corpulence :

Svelte

Taille :

170 cm

Cheveux :

Noirs légèrement grisonnants

Barbe :

Épaisse

Yeux :

Fatigués

## Signature Visuelle

* manteau de voyage
* vêtements simples
* bottes usées
* épée simple

Unique symbole royal :

Couronne de laurier en or.

Aucune armure royale.

Aucune cape de roi.

Aucun ornement extravagant.

## Évolution Visuelle

Début :

Voyageur solitaire.

Milieu :

Roi errant.

Fin :

Roi légendaire.

L'évolution reste élégante et sobre.

# 6. Billy

Billy est le premier compagnon.

Billy est un **Grand Loup** (canon D-A01), jamais un simple chien.

Au début :

* maigre
* sale
* affamé

Après le prologue :

* plus robuste
* fidèle
* protecteur

Toujours conserver :

une légère étrangeté dans son regard.

Le joueur doit sentir qu'il cache un secret.

---

# 7. Tobo

## Direction

Tobo n'est pas un robot.

Tobo est une relique.

Une machine divine.

Une technologie oubliée.

## Apparence

Formes :

* anneaux flottants
* géométrie impossible
* lumière cyan
* lumière violette

Inspirations :

* artefact cosmique
* intelligence ancienne
* observateur

Le joueur ne doit jamais savoir immédiatement :

si Tobo est une machine,
une divinité,
ou autre chose.

# 8. Le Royaume

## Concept

Le Royaume n'est pas une capitale.

Le Royaume est un refuge.

Un petit royaume fortifié en reconstruction.

## Début du jeu

90 % ruines

10 % vivant

## Fin du MVP

50 % ruines

50 % vivant

Le joueur doit voir physiquement :

* les bâtiments revenir
* les habitants arriver
* l'espoir renaître

# 17. Règle d'Or

Chaque asset produit doit répondre à cette question :

"Est-ce que cet élément appartient à un rêve mourant que l'on tente de réparer ?"

Si la réponse est non,

l'asset doit être rejeté.
