// IMPORTANT

Ce document définit la structure de progression MVP.

Les quêtes, objectifs, récompenses, personnages et étapes narratives
sont susceptibles d'être modifiés lors des futures itérations.

L'objectif principal de ce document est de fournir une base cohérente
pour l'implémentation du système de quêtes et de progression.

// Quest Philosophy

Les quêtes d'IdleKing ont pour objectif de :

- faire progresser le récit
- faire découvrir le Monde Rêvé
- introduire les mécaniques
- reconstruire le Kingdom
- guider le joueur

Les quêtes ne doivent pas devenir des tâches artificielles de type
"tuer X ennemis" ou "collecter Y objets" sans justification narrative.

Même lorsqu'une quête possède un objectif gameplay,
celui-ci doit être justifié par le récit.

# QUEST DATABASE — IdleKing (V2)

Foundation V2 — Subject to Future Iterations

This document defines the MVP quest structure for:

* Prologue
* Chapter I — Era Funèbre
* Chapter II — Era Glaciaire

The purpose of this document is to provide a stable implementation foundation.

Quest objectives, rewards, NPCs, narrative beats and progression may evolve during future iterations.

---

# Quest Philosophy

Quests in IdleKing exist to:

* advance the narrative
* discover the Dream World
* introduce mechanics
* rebuild the Kingdom
* guide the player through progression

Quests should avoid becoming artificial tasks disconnected from the story.

Whenever possible, gameplay objectives should support narrative objectives.
