# Philosophy

L'interface d'IdleKing doit être :

* lisible
* élégante
* discrète

L'UI ne doit jamais voler l'attention du monde.

Le joueur joue dans le Monde Rêvé.

Pas dans l'interface.

# Cyan Rule

Le cyan est rare.

Le joueur doit immédiatement comprendre :

"Quelque chose d'important se produit."

Le cyan est réservé à :

* Tobo
* Kaléidoscope
* Time Gate
* Fleuve de Vie
* phénomènes liés à Créa
* phénomènes liés au Temps

Interdiction :

Utilisation décorative.

---
