# Philosophy

Les personnages d'IdleKing ne sont pas des habitants d'un monde fantasy classique.

Le Monde Rêvé est un espace façonné par les souvenirs, les émotions et les âmes.

Chaque personnage doit sembler être :

* une mémoire
* un vestige
* un rêve
* une émotion incarnée

Même les personnages les plus humains doivent conserver une légère étrangeté.

Le joueur doit constamment ressentir :

"Quelque chose n'est pas totalement réel ici."

# GOLDEN RULE

Aucun personnage ne doit être conçu comme un simple archétype fantasy.
