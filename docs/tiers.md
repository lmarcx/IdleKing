# 🧬 TIERS — Idle King

## Structure

Age = Tier

5 Ages = 5 Tiers

Tier I  → World 1–10
Tier II → World 11–20
Tier III → World 21–30
Tier IV → World 31–40
Tier V → World 41–50

---

## Effets d’un Tier

Chaque Tier :

- Augmente la difficulté globale
- Augmente ilvl max loot
- Augmente reward multiplier
- Débloque nouveaux Buildings
- Débloque Chapitres (4 par Tier)

---

## Unlock Tier II

Conditions :

- WorldLevel ≥ 20
- Building spécifique requis
- Coût en Kingamas

---

## Philosophie

Tier = palier majeur de puissance.

Le passage de Tier doit être :
- Marquant
- Coûteux
- Structurel
