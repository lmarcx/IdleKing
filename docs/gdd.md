# 👑 Idle King — Game Design Document

## 🎮 Genre

Idle RPG / Progression exponentielle contrôlée
Dark fantasy pixel 2D

Inspirations :
- Paperclips (Farm / Scaling / World building)
- Tales of Win (Contenu instancié + progression)

---

# 🟢 MVP Scope

## Player max level : 50
## World max level : 50
## 4 Biomes
## 10 Expéditions
## 3 Leaderboards

---

# 🔵 Core Loops

### Loop 1 — Leveling
Histoire → Donjons → Ressources → Buildings → XP

### Loop 2 — Risk
Expédition → Risk Loadout → Boss → Kingamas

### Loop 3 — Optimization
Loot → Upgrade → POWER → Speedrun

---

# 🟣 Systems

- POWER engine
- Loot deterministic
- Boss phases
- Risked items
- Kingamas wallet
- Leaderboards

---

# 📈 Progression

- Player Level = secondaire endgame
- World Level = structure de progression
- Buildings = moteur de meta progression

Progression cible :

- Lvl 50 ≈ 15h actif
- World 50 ≈ dépend stratégie buildings
