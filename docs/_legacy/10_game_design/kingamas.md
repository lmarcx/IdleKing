# 👑 KINGAMAS — Idle King

## Rôle

Meta-currency centrale du jeu.
Représente la domination du joueur sur le monde.

Utilisée pour :
- Buildings avancés
- Déblocage Tier II
- Systèmes endgame
- Futur PvP
- Renaissance

---

## Acquisition

Les Kingamas sont obtenus via :

- ✅ Win expedition (RNG dans plage fixe)
- ✅ Repeatable quests
- ✅ Conversion de ressources

Règle absolue :
LOSE = 0 Kingamas

---

## Philosophie

Les Kingamas doivent :

- Être rares
- Être risqués
- Être prestigieux

Pas d’auto génération offline.
Pas de gain passif infini.

---

## Inflation Control

- Plage fixe par niveau d’expédition
- Hard cap par semaine
- Dépendance au skill / POWER

---

## Long Term

- PvP stakes
- Classement mensuel
- Déblocage contenu Cieux / Abysses
- Renaissance scaling
